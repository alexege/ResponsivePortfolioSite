Slick Notes

Slick Notes is a note taking app built in Django. The goal of this project is to allow the end user to document anything they wish in one convenient place.

Users may sign up to create their own notebooks with notes only accessible to them.

I plan to one day integrate a sharing option that would allow quick and easy collaboration between note-takers.

For now, users may create notes and categorize them as they wish.

Features

  • Ability to add Category
  • Add Note
  • Add Comment
    • Add plaintext comment
    • Add code snippet
    • Add image
    • Add pdf
    • Add video
  • Search for specific note or content
  • Drag and Drop abilities for both notes/notebooks
  • Toggle publicity of each category.

Drag and drop notes as well as categories

Back Story

I started this app with a simple goal. I quickly found I wanted to add the ability to do everything without refreshing the page. Once I figured out how to do that, I had to apply that to each function of this project.
