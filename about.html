Slick Notes

Slick Notes is a note taking app built in Django. The goal of this project is to allow the end user to document anything they wish in one convenient place. Users can create notebooks with categories to classify their notes. Once created, a notebook can be made public and shared with the community.

User has the option to login with an existing account or login as guest.

User can register a new account.

I have implemented a share option that would allow quick and easy collaboration between note-takers.

Users may create notes and categorize them as they wish.

Features

  1. Add a Notebook
  2. Add a Category
  3. Add a Note
  4. Add a Comment
    • Add plaintext comment
    • Add code snippet
    • Add image
    • Add pdf
    • Add video
  5. SearchBar: Search for specific note or content
  6. Drag and Drop functionality for both notes/notebooks
  7. Toggle publicity of each category.

Summary:

I started this app with a simple goal: create a simple note taking app that I would actually use. I quickly found I wanted to add the ability to do everything without refreshing the page. Once I figured out how to do that, I had to apply that to each function of this project. You will notice the app updates instantaneously avoiding all page reloads. This was acheived using Ajax and could also be done by implementing a front-end framework which I plan to do in the future.

Blur Read

Blur Read is an anonymous social media platform designed to allow users to speak thier mind without feare of ridicule or criticism. This project utilizes a random character generator to create unique strings which stand in for traditional usernames to keep the end user anonymous. Users will start off with a generic icon associated with thier account. As their posts gain traction, they will unlock new tiers, each with their own custom icon. If a user reaches a specific popularity level, they will be granted the ability to disclose who they are with thier own custom username.

Standard user login with validation.

This is the dashboard. This is where users will be able to see their feed.

Features

  1. Add a new post
  2. Anonymous posting
  3. Report a post
  4. Hide a post
  5. Mark as positive
  6. Mark as negative
  7. Click on category name to see all posts filtered to specified category

Summary:

This was a fun project that my friend Kian Chachi and I developed. We joined forces to create a unique anonymous social media platform. We wanted to provide users with the ability to voice their opinion without the worry of curating their post before posting. In today's society, we tend to restrict ourselves in feare of ridicule or embarassment. This website was designed with the purpose of removing the link to an individual to focus more on the content itself and not the person making the post.
